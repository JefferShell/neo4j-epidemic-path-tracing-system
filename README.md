## 项目简介

这个项目是一项基于Web的论坛平台，旨在为用户提供一个交流学术问题的平台。项目主要功能包括：

- 用户注册登录系统
- 浏览帖子列表并收藏/点赞帖子
- 查看帖子详情并评论
- 管理帖子分类
- 搜索帖子
- 爬取考研信息页面
- 与chatGPT交互，获得答案并发布为帖子

## 技术架构

项目采用以下技术架构：
- 前端：React
- 后端：Node.js
- 数据库：MySQL

1. 使用项目，请先在本地安装所有依赖项。
